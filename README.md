## 项目介绍

基于 Vue 3 + Spring Boot + COS + WebSocket 的 **企业级智能协同云图库平台**

该项目核心功能可以分为以下四大类

1. 所有用户都可以在平台上公开上传和检索图片素材，寻找需要的图片。可用于表情包网站、设计素材网站、壁纸网站等

![](https://a1.boltp.com/2026/01/13/696604bfe1376.png)

2. 管理员可以上传、审核以及管理图片，并对系统中的图片进行分析

![](https://a1.boltp.com/2026/01/13/696605ce8dbda.png)

3. 对于个人用户，可以把图片上传到自己的私有空间进行批量管理、检索、编辑和分析（***待实现***）
4. 对于企业，可以开通团队空间并且可以邀请成员共享图片，还能**实施协同编辑图片**，提高团队协作效率（***待实现***）
